Distributions

The ASxxxx Cross Assemblers are available for Intel based PC's in the following forms:

  • ASxxxx Cross Assemblers as a zipped file (750K)
    (use 'pkunzip -d asv2p0.zip' for extraction)

  • ASxxxx Cross Assemblers as a self extracting zip file (765K)
    (use 'asv2p0.exe -d' for extraction)

The ASxxxx Cross Assemblers are also available for PDP-11's in a single logical disk, a logical disk containing 5 RX02 logical disks, or as individual RX02 logical disks. Binary transfer must be used.

  • ASxxxx Cross Assemblers in a single RT-11 logical disk file (1900K)

  • ASxxxx Cross Assemblers as a multi-level logical disk (2500K)

    1. Disk1 of ASxxxx Cross Assemblers (494K)
    2. Disk2 of ASxxxx Cross Assemblers (494K)
    3. Disk3 of ASxxxx Cross Assemblers (494K)
    4. Disk4 of ASxxxx Cross Assemblers (494K)
    5. Disk5 of ASxxxx Cross Assemblers (494K)
