Distributions

The ASxxxx Cross Assemblers are available in the following forms:

  • ASxxxx Cross Assemblers as a zipped file (2700K)
    (use 'pkunzip -d asv2p1.zip' for extraction with MS-DOS)
    (use 'unzip -L -a asv2p1.zip' for extraction with Linux)

  • ASxxxx Cross Assemblers as a self extracting zip file (2715K)
    (use 'asv2p1.exe -d' for extraction with MS-DOS)

The ASxxxx Cross Assemblers are also available for PDP-11's in a single logical disk, a logical disk containing 6 RX02 logical disks, or as individual RX02 logical disks. Binary transfer must be used.

  • ASxxxx Cross Assemblers as a multi-level logical disk (3000K)
